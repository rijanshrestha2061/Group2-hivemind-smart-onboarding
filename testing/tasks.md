
## Task 1 — Find what to do on Day 1 and track progress
**Ties to:** H1 (Guided first steps + progress)
**Success looks like:** user can identify their top priority task within the first minute, and later say at a glance what's done and what's next.

## Task 2 — Find the correct, current version of a document
**Ties to:** H2 (Centralised information / single source of truth)
**Success looks like:** user can locate a document and confirm it's the current version without asking a colleague.

## Task 3 — Find who to contact, and understand an AI recommendation
**Ties to:** H3 (Trustworthy, contextual support)
**Success looks like:** user can identify the right contact for a task within 30 seconds, and correctly explain the source/last-updated date behind an AI-suggested next step.

## Task 4 — Recognise role-specific content
**Ties to:** H4 (Role-specific content over generic training)
**Success looks like:** user notices and can describe how the content shown is tailored to their role, rather than generic.
